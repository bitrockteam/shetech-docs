Finora creando dei componenti e componendoli nella nostra applicazione, React si limita a renderizzare le diverse parti in modo statico. Scrivendo negli input qualcosa non scateneremo dei cambiamenti nell'applicativo permanenti ma potremo solo influenzare o leggere gli elementi HTML sulla pagina. Per far cambiare il contenuto delle pagine, React si aspetta che i props di un componente cambino; e per far cambiare i props non basta cambiare le variabili dentro o fuori una funzione, dovremo invece cambiare lo __stato__ del nostro componente genitore per far si che React noti il cambiamento lo applichi a cascata ai suoi figli.
