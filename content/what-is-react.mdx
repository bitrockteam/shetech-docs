React è una libreria per creare interfaccie utente, in particolare per il web, basata sui concetti di stato e di componente. Nella sostanza è un modo per scrivere dell'HTML dentro la pagina e farlo cambiare in seguito a interazioni e altri eventi; React come altre librerie nasce per semplificare la creazione di applicazioni web e interazioni complesse. Dove prima era necessario cambiare la pagina elemento per elemento, adesso è possibile descrivere come dovrebbe apparire l'HTML e il CSS dopo l'interazione dell'utente o eventi come il recupero di informazioni da un server; React provvede a cambiare l'aspetto della pagina per noi.

L'intuizione alla base di React sta nel fatto che cambiare l'HTML sulla pagina è molto più lento che cambiare un oggetto in memoria. React conserva quindi una versione virtuale della nostra pagina (o virtual DOM); quando lo stato o i props di cui parleremo cambiano, crea una nuova versione virtuale, la confronta con la precedente trovando i cambiamenti e quindi il minor numero possibile di cambiamenti da fare alla pagina per non causare eccessivo lavoro grafico, molto più pesante rispetto al confronto nella memoria.
