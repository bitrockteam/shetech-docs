Ogni volta che utilizziamo React teniamo in considerazione che stiamo aumentando non di poco la complessità di un progetto. Vi sono casi in cui la interfaccia che vogliamo creare è cosi semplice che React potrebbe anzi complicarci la vita, oppure potremmo usare applicativi come WordPress dove, essendovi già HTML e CSS prodotto dal server, non ci serva aggiungere eventi complessi e altre features.

Ci rivolgiamo a queste librerie soprattutto per casi come:

- progetti che riutilizzino una serie di moduli comuni
- applicativi con interazioni simili a quelle di applicazioni native
- applicativi basati su API e dinamici

Alcuni casi meno probabili:

- Blog (es. WordPress)
- Siti vetrina
- eCommerce (es. Magento)
