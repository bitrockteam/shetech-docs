In alcuni casi ci viene comodo uscire dalla nostra funzione prima del necessario, quando magari dobbiamo mostrare un errore; questo è il classico caso in cui possiamo usare if nella nostra funzione. Le clausole condizionali funzionano correttamente al di fuori del blocco di JSX:

```jsx
    function grazie ({completatoCorrettamente = true}) {
        if (!completatoCorrettamente) {
            return (
                <span className="errore">Qualcosa è andato storto</span>
            )
        }

        return (
            <span>Grazie per aver partecipato al sondaggio!</span>
        )
    }
```
